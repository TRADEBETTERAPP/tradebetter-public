# Contributing to TradeBetter

Thank you for helping improve TradeBetter. This repository is the public interface for reporting bugs, requesting features, and asking questions. It does **not** accept pull requests ‚Äî all source code lives in private repositories.

---

## How to Write a Good Bug Report

A clear bug report gets resolved faster. Include:

1. **Summary** ‚Äî One sentence describing what's broken.
2. **Steps to reproduce** ‚Äî Exact, numbered steps to trigger the bug. Include URLs, button names, and sequences.
3. **Expected behaviour** ‚Äî What should have happened.
4. **Actual behaviour** ‚Äî What actually happened. Include error messages verbatim.
5. **Environment** ‚Äî OS, browser (with version), wallet provider (MetaMask, Phantom, Privy embedded), network (Polygon, Base), and app version if visible.
6. **Evidence** ‚Äî Screenshots, screen recordings, console logs, or network traces. Redact any sensitive data.

**Example of a good summary:**
> "Deposit modal shows 'undefined' for USDC balance when connected via MetaMask on Base"

**Example of a bad summary:**
> "deposits broken"

---

## How to Propose a Feature

1. **Problem** ‚Äî What problem are you trying to solve? Be specific about your workflow.
2. **Proposed solution** ‚Äî Describe what you'd like to see. Mockups, sketches, or references to other products are welcome.
3. **Alternatives considered** ‚Äî What else could solve this? Why is your proposal better?
4. **Impact** ‚Äî Who benefits? How often would this be used?

## What Happens After Submission

| Stage | What it means |
|-------|---------------|
| `triage` | Your issue has been received. The team will review it within a few business days. |
| `accepted` | The issue is valid and has been scheduled for work. |
| `in-progress` | A team member is actively working on a fix or implementation. |
| `shipped` | The fix or feature has been deployed. The issue will be auto-closed. |
| `wontfix` | The team has decided not to address this, with an explanation in the comments. |
| `duplicate` | This is a duplicate of an existing issue. A link to the original will be provided. |

---

## What NOT to Post

**Do not post any of the following in issues or comments:**

- Wallet private keys or seed phrases
- API keys, tokens, or credentials
- Security vulnerabilities or exploit details (see [SECURITY.md](SECURITY.md))
- Personal identifying information of other users
- Abusive, threatening, or harassing content

Issues containing any of the above will be immediately removed.
